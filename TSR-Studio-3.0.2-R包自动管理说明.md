# TSR Studio 3.0.2：R 包自动检查与按需安装

## 使用方法

1. 打开“基础分析”。
2. 点击“检测运行环境与 R 包”。
3. 软件按模块显示“已就绪”或“缺少包名”。
4. 可点击某模块的“安装缺失包”，也可点击“一键补齐全部缺包”。
5. 安装完成后软件自动复检；对应模块全部就绪后再运行分析。

## 自动管理范围

- 基础统计：DESeq2、limma、edgeR、jsonlite。
- GO/KEGG/GSEA 与注释：clusterProfiler、org.Mm.eg.db、org.Rn.eg.db、org.Hs.eg.db。
- 网络分析：WGCNA。
- 高级整合：MOFA2、mixOmics、basilisk。

## 跨平台与安全规则

- macOS 与 Windows 使用同一依赖清单和检测逻辑。
- R 包安装在 TSR Studio 的用户数据目录，不要求管理员权限，不写入应用安装目录。
- 软件分析进程自动读取同一用户包库。
- 只安装缺失项，不重复安装已有包，不自动升级现有包。
- 缺少 R 本体、网络失败、包编译失败或版本不兼容时明确阻断并保留日志，不生成伪结果。
- 首次安装 Bioconductor 大型包可能耗时较长，需要保持联网并等待操作完成。
